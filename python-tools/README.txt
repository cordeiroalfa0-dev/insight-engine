Pré-requisitos:
  pip install pillow

1) Baixar imagens das ROMs detectadas (de uma vez)
   python download_rom_images.py "C:\caminho\roms" "C:\caminho\imagens"

2) Melhorar nitidez/cores/contraste das imagens
   python enhance_images.py "C:\caminho\imagens" "C:\caminho\imagens_hd"
